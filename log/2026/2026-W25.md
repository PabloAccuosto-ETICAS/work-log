## Eticas Platform

- **2026-06-16: ADR 0003 written — target architecture reframed as a single parametrised model.** Reviewing the Fullcircle proposal, established that the target is not a choice between architecture options but one parametrised model: a fixed Eticas-hosted **control plane** + an **execution plane whose location is an engagement parameter**, with **A (Eticas-side execution) and C (client-side runner) as presets**, selected per engagement by audit type / threat model. Driver: black-box and adversarial auditing cannot run from inside the client perimeter without breaking validity. Committed [`docs/decisions/0003-execution-plane-location-parameter.md`](https://github.com/Eticas-AI/ai-platform/blob/main/docs/decisions/0003-execution-plane-location-parameter.md); STATE open-decision on target architecture repointed to it.
